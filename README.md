# ChatterBox 💬
A real-time chat application built with Spring Boot and WebSocket.

## Features
- Real-time messaging with WebSocket + STOMP
- Join/Leave notifications
- Chat history with MySQL
- Dark themed UI

## Tech Stack
- Java 21, Spring Boot 4
- WebSocket, STOMP
- MySQL, Spring Data JPA
- HTML, CSS, JavaScript

## Setup
1. Clone the repo
2. Create MySQL database: `CREATE DATABASE chatterbox;`
3. Copy `application.properties.example` to `application.properties`
4. Update your MySQL credentials
5. Run `FirstDemoApplication.java`
6. Open `http://localhost:8080`
